Модуль калькулятора з підтримкою збереження історії та роботи з пам'яттю.
 
Цей модуль надає функціонал для виконання базових арифметичних операцій з автоматичним збереженням історії у файл `History.txt`. 
Також реалізовано клас `Memory`, який дозволяє працювати з проміжними результатами в пам'яті.
 
Класи:
--------
1. Calculator:
   Базовий клас для виконання математичних операцій. Історія обчислень зберігається у файл `History.txt`.
 
2. Memory:
   Наслідуваний від `Calculator` клас для роботи з пам'яттю. Дозволяє зберігати, додавати, віднімати та очищувати значення в пам'яті.
 
Методи:
-------
Клас `Calculator`:
    - Save(value): Зберігає текстове представлення операції у файл `History.txt`.
    - Add(a, b): Виконує додавання двох чисел.
    - Subtract(a, b): Виконує віднімання другого числа від першого.
    - Multiply(a, b): Виконує множення двох чисел.
    - Divide(a, b): Виконує ділення двох чисел. Обробляє ділення на нуль.
    - Mod(a, b): Обчислює залишок від ділення першого числа на друге.
    - SquareRoot(number): Обчислює квадратний корінь з числа.
    - Exponentiate(a, b): Підносить перше число до степеня другого.
    - GetHistory(): Читає та виводить історію обчислень з файлу `History.txt`.
 
Клас `Memory`:
    - Add(value): Додає значення до поточного в пам'яті.
    - Subtract(value): Віднімає значення від поточного в пам'яті.
    - Clear(): Очищає пам'ять, встановлюючи значення на 0.
    - Read(): Читає поточне значення з пам'яті.
 
Деталі реалізації:
------------------
1. **Історія**:
    - Усі результати операцій автоматично записуються в файл `History.txt`.
    - Формат запису: `<операція> = <результат>`. У разі помилки (наприклад, ділення на нуль) також записується відповідний рядок.
 
2. **Пам'ять**:
    - Значення в пам'яті доступне лише в рамках класу `Memory`.
    - Запис до файлу відбувається при кожній зміні значення в пам'яті.
 
Приклади використання:
----------------------
>>> calc = Calculator()
>>> calc.Add(3, 5)
8
>>> calc.SquareRoot(16)
4.0
>>> calc.GetHistory()
3 + 5 = 8
Square root of 16 = 4.0
 
>>> mem = Memory()
>>> mem.Add(10)
10
>>> mem.Subtract(3)
7
>>> mem.Clear()
0
