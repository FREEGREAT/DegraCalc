Цей модуль містить клас BasePlot для створення, 
відображення та збереження графіків 
за допомогою бібліотеки Matplotlib.
 
Клас BasePlot надає базову функціональність 
для роботи з графіками, включаючи методи 
для їх відображення на екрані
та збереження в різних 
форматах файлів (наприклад, PNG, PDF, JPG).

--------
- show(): Відображає графік у вікні.
